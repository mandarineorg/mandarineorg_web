A minimalist,
decorator-driven,
MVC framework for Deno.

Typescript, all the way down

Mandarine & all its sub-projects run on Deno, using Typescript as its main language, giving you better coding environments and Javascript's latest features.

Straightforward

The use of decorators & the simplicity of a single entry-point application makes development faster, easier and more readable.

All-in-one

Mandarine along with its 4 different cores: Core, Security, Data & MVC bring a compacted, SOLID oriented solution for complex applications.

Deno + Mandarine + Typescript

The internal features Deno brings such as importing by URL + Mandarine's syntax & tools +
Typescript objected-oriented coding style is all you will ever need for a Mandarine-powered application
